## 🎮 Features

- 🎯 Choose between ✊ Rock, 📃 Paper, and ✂ Scissors
- 👤 Enter your name before playing
- 🔢 Live score updates
- 🏆 First to 5 points wins
- 🔁 Play again button to restart
- 🎨 Responsive and colorful design
